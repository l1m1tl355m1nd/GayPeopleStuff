Instruction Set Architecture

Updated reference for the custom 32-bit ISA, using the assembly syntax defined in #ruledef from customasm. Covers immediate and register storage, arithmetic, and branch instructions.

03

Register Store / Output

ASM strr {reg2}, {reg1}   |   stro {decoder}, {reg1}
FieldBitsDescription
Opcode000105-bit opcode
Reg15Source register
Reg25Destination (for register store), unused for OUT
Write Enable1Enable register write
Switch150 = register move; 1–14 = output (15-bit pin addressing)
Switch — Set to 0 for a register-to-register move (strr). Any non-zero value activates the output path (stro), with the value used as a 15-bit decoder address for pin output.

Jump / Branch with Link

ASM jmp {switch}, {loc_reg}, {answr_reg}
FieldBitsDescription
Opcode001005-bit opcode
Answer Reg5Register to save return address into
Loc Reg5Register holding jump target address
Switch1Return address enable (linked jump / call)
Unused16—
Note — The jump condition is evaluated from the flags set by a previous com* instruction. When switch is set, the current PC is saved to answr_reg before jumping — enabling subroutine calls.
